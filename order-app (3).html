    📊 เชื่อมต่อ Google Sheet

    เก็บข้อมูลออเดอร์ใน Google Sheet ทุกคนในทีมจะเห็นข้อมูลเดียวกันทันที ใช้งานฟรี 100%

    1
    สร้าง Google Sheet ใหม่
    ไปที่ sheets.google.com → สร้าง Spreadsheet ใหม่
    2
    เปิด Apps Script
    เมนู Extensions → Apps Script → ลบโค้ดเดิม → วางโค้ดจากไฟล์ apps-script.js → Save
    3
    Deploy เป็น Web App
    กด Deploy → New deployment → เลือก Web app → Execute as: Me → Who has access: Anyone → Deploy → Authorize → Copy URL
    4
    วาง URL ด้านล่าง
    วาง URL ที่ได้จาก Step 3 แล้วกด "บันทึกและทดสอบ"
